# KBHS Teacher Notes

## Description
Notes management system for Kasikeu Boys High School teachers.

## Installation
To install the necessary dependencies, run the following command:

```bash
npm install
```

## Usage
To start the development server, run:

```bash
npm run dev
```

You can access the application at [http://localhost:3001](http://localhost:3001).
